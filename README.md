# 登山クイズ
サイトURL https://yama-quiz.herokuapp.com/
開発期間 2022.02~

## サービス概要
登山を楽しみたいハイカーに対して、 クイズを解きながら楽しく登山に関する知識をつけられるサービス

## ユーザー
登山の知識をつけるのが好きな人
登山に関心を持ったが何から勉強すればいいかわからない人

## サービス目的
登山を楽しむには専門知識が不可欠。クイズを通して知識をつけてもらい、登山ライフに活かしてもらう
